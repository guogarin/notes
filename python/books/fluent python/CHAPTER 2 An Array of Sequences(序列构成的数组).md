# 1. 列表的`+`和`*`运算符
## 1.1 `+`和`*`运算符 分别完成算么工作？
(1) `+` 将返回一个新列表，此列表由 `+`两侧的列表拼接而成；
```python
l1 = [1, 2, 3]
l2 = [4, 5, 6]

l3 = l1 + l2

print(f"l3: {l3}")
```
运行结果：
```
l3: [1, 2, 3, 4, 5, 6]
```
(2) `list * n` 将返回一个新列表，此列表由 `n`份`list`列表拼接而成：
```python
l1 = [1, 2, 3]

l3 = l1 * 3

print(f"l3: {l3}")
```
运行结果：
```
l3: [1, 2, 3, 1, 2, 3, 1, 2, 3]
```

## 1.2 `+`和`*`运算符会修改原列表吗？
`+`和`*` 都不会修改原列表， 而是构建一个全新的序列来存放结果：
```python
l1 = [1, 2, 3]
l2 = [4, 5, 6]

l3 = l1 + l2

print(f"l3: {l3}")

print(f"id(l1) : {id(l1)}")
print(f"id(l2) : {id(l2)}")
print(f"id(l3) : {id(l3)}")
```
运行结果：
```
l3: [1, 2, 3, 4, 5, 6]
id(l1) : 1528544639296
id(l2) : 1528544628032
id(l3) : 1528544624896
```
