# 3. 说说 HTTP 状态码及其含义
&emsp;&emsp; HTTP 状态码由三个十进制数字组成，第一个十进制数字定义了状态码的类型。响应分为五类：信息响应(100–199)，成功响应(200–299)，重定向(300–399)，客户端错误(400–499)和服务器错误 (500–599)：

| 分类  | 分类描述                                       |
| ----- | ---------------------------------------------- |
| `1**` | 信息响应，服务器收到请求，需要请求者继续执行操作   |
| `2**` | 成功响应，操作被成功接收并处理                     |
| `3**` | 重定向，需要进一步的操作以完成请求             |
| `4**` | 客户端错误，请求包含语法错误或无法完成请求     |
| `5**` | 服务器错误，服务器在处理请求的过程中发生了错误 |
